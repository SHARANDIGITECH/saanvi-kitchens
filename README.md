# Saanvi Kitchens

Create a modern, responsive 4-page website for a "Home Kitchen" food business using pure HTML, CSS, and JavaScript (no frameworks).

The design should be premium, elegant, and food-focused with a dark theme and yellow accent colors.

🔹 Theme & Style:

* Dark background (#0f0f0f or similar)
* Yellow accent color (#FFD700 or similar)
* Clean modern typography (sans-serif)
* Smooth animations and hover effects
* High-quality food images across all sections
* Fully responsive (mobile + desktop)

🔹 Pages Required:

1. Home Page (index.html):

* Hero section with a delicious food background image
* Headline: “Delicious Homemade Food Delivered to You”
* Subtext + CTA button (Order Now / View Menu)
* Featured dishes section (3–6 items with images)
* Why Choose Us section (Fresh, Hygienic, Homemade, Fast Delivery)
* Testimonials (2–3 customer reviews)
* CTA banner section
* Footer with links and social icons

2. About Us Page (about.html):

* Story of the home kitchen (homemade, love, hygiene)
* Chef or founder introduction with image
* Mission & values
* Images of kitchen / cooking process

3. Menu Page (menu.html):

* Categorized menu (Breakfast, Lunch, Dinner, Snacks, Desserts)
* Each item with image, name, description, and price
* Add hover effects on cards
* Optional: filter buttons (JS-based)

4. Contact Us Page (contact.html):

* Contact form (Name, Phone, Message)
* Google Maps embed
* Address, phone, email
* Social media links

🔹 Navbar:

* Sticky navigation bar
* Logo on left, menu on right
* Links to all 4 pages
* Mobile hamburger menu (JS toggle)

🔹 Footer:

* Quick links
* Contact info
* Social icons
* Copyright

🔹 JavaScript Features:

* Mobile menu toggle
* Smooth scrolling
* Simple form validation
* Optional menu filtering

🔹 UI Enhancements:

* Card-based layout for dishes
* Image overlays and hover zoom effects
* Subtle animations (fade-in, slide-up)
* Section spacing and clean layout

🔹 Assets:

* Use relevant high-quality food images (free stock style)
* Ensure images match home-cooked food theme

🔹 Goal:
The website should feel premium, trustworthy, and appetizing—encouraging users to order food immediately.

🔹 Output:
Generate complete working code with:

* Separate HTML files (index, about, menu, contact)
* One CSS file (style.css)
* One JS file (script.js)
* Clean folder structure
